Frontend Admin (Editor)

Interfaz administrativa para crear y editar contenido educativo.
Aqui se encuentran los archivos HTML, CSS y JavaScript que permiten a los administradores:
- Crear y editar materias, cursos y lecciones
- Editor Markdown + LaTeX
- Subida de imagenes o figuras para las diferentes lecciones
- Gestion de ejercicios mediante formularios

Es una interfaz avanzada, separada del frontend de usuario, para poder subir, editar y eliminar contenido.
